## 新增需求

### 需求：验证文件存在于静态资源目录

web_query_service 的静态资源目录必须包含验证文件。

#### 场景：验证文件位置正确

- **当** 查看 `web_query_service/static/` 目录
- **那么** 必须存在文件 `e117c5d282297a44d17e18afb8c357f3.txt`

---

### 需求：验证文件内容正确

验证文件的内容必须为指定的验证码。

#### 场景：文件内容匹配

- **当** 读取 `e117c5d282297a44d17e18afb8c357f3.txt` 文件
- **那么** 文件内容必须为 `b0455d9c5df12acae5ee0d23c065000c5479bb16`

---

### 需求：构建后验证文件可访问

构建部署后，验证文件必须可通过根路径访问。

#### 场景：部署后文件可访问

- **当** 执行 `pnpm run build` 构建项目
- **那么** `public/e117c5d282297a44d17e18afb8c357f3.txt` 文件必须存在
- **并且** 文件内容必须与源文件一致
