## ADDED Requirements

### 需求：同步按钮可见性

卡片列表工具栏中的"同步到云端"按钮必须仅在云端同步 API 已配置时显示。

#### 场景：同步 API 已配置

- **当** 用户已配置云端同步 API URL 和 API Key
- **那么** 卡片列表工具栏中必须显示"同步到云端"按钮
- **并且** 按钮必须使用上传/同步图标以区别于其他操作按钮

#### 场景：同步 API 未配置

- **当** 用户未配置云端同步 API URL 或 API Key
- **那么** 卡片列表工具栏中禁止显示"同步到云端"按钮
- **并且** 不占用工具栏空间

---

### 需求：触发云端同步

点击"同步到云端"按钮必须执行与"数据管理 → 云端同步 → 立即同步"完全相同的同步操作。

#### 场景：同步成功

- **当** 用户点击"同步到云端"按钮
- **那么** 按钮必须进入 loading 状态（显示加载动画，禁止重复点击）
- **并且** 必须调用 `execute_sync_cmd` Tauri 命令执行同步
- **并且** 同步成功后必须显示成功提示消息（包含同步的数据统计）
- **并且** 按钮必须恢复为正常状态

#### 场景：同步失败

- **当** 用户点击"同步到云端"按钮
- **并且** 同步过程中发生错误（网络错误、认证失败等）
- **那么** 必须显示错误提示消息
- **并且** 按钮必须恢复为正常状态

#### 场景：同步进行中禁止重复触发

- **当** 同步操作正在进行中
- **那么** "同步到云端"按钮必须处于 loading/禁用状态
- **并且** 禁止再次触发同步
